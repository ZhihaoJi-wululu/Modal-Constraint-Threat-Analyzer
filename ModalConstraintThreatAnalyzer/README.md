# Modal Constraint Threat Analyzer

Browser-based formal modeling workspace plus a Python rule engine for modal-constraint social engineering threat detection.

## Run

Use Python 3.10+ from this folder:

```bash
python run_server.py
```

Then open:

```text
http://127.0.0.1:8000
```

The backend uses only the Python standard library. Do not start the service automatically during development unless the user explicitly asks.
